# Cellular Automation

This python program uses PyQt5 to generate Cellular Automation which you can tweak.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install opencv-contrib-python
pip install PyQt5
```
